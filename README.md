# Просто скопируйте эту команду и вставьте в терминал вашего сервера (работает на Ubuntu/Debian/CentOS):
    wget -O gotg.sh https://raw.githubusercontent.com/kir460/gotg/main/gotg.sh
chmod +x gotg.sh
sudo ./gotg.sh
После установки управление скриптом командой:
    gotelegram
1.11 Далее процесс выглядит так: 
Закрытый чат и помощь 
1 шаг. Нажимаем 1 (Установить / Обновить прокси). 
2 шаг. Выбираем сайт маскировки (если сервер заграницей то лучше .com 
домен, если сервер .ru то лучше сайт с доменом .ru 
3 шаг. Выбираем порт (рекомендуется 443)
1.12 Импорт конфигурации возможен 2 вариантами: 
Закрытый чат и помощь 
1) Отсканируйте камерой телефона QR код и перейдите в приложение 
Telegram для импорта конфигурации
2) Скопируйте из Termius и перейдите по ссылке типа: 
tg://proxy?server= и импортируйте ключ. 
Внимание !Все у кого есть ключ могут подключаться без ограничений, 
будьте аккуратны с публикацией ключа.
